# Tablea Sales and Customer Dashboard

## Overview

This project involves the creation of two comprehensive dashboards in Tableau, designed to visualize key performance indicators (KPIs) for a company's sales and customer metrics. 
The dashboards provide insights by comparing the current year's (CY) performance with the previous year's (PY) data. The data used in this project was cleaned and prepared using
Excel's Power Query Editor.

## Features

### Key Visualisations

  * Total Sales
  * Profit
  * Quantity
  * Total Orders
  * Total Customers
  * Sales Per Customers

### Year based Comparisions 

The dashboards enable comparison of key metrics between the current year (CY) and the previous year (PY) to identify trends and performance changes.
